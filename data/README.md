# Datasets

- ```dataset_training.jsonl```: 訓練資料
- ```dataset_testing.jsonl```: 測試 ```load_model.py```
- ```prediction_testing.txt```: ```dataset_testing.jsonl``` 的測試結果
- ```frequency_distribution.png```: ```dataset_training.jsonl``` 的資料分布
